PictureSelectionViewController

class PictureSelectionViewController : UIViewController, UIImagePickerControllerDelegate

A resusable view controller for taking an image to use in a dataset

Author

Jake Trimble

Copyright

© 2019 Hello World. All rights reserved.

Date

Created 4/22/19

Important

  • Either the stack (For mutliple images) or imageTitle (For one image) properties must be set before presentation
  • Provide the stack property with a list of image names or the imageTitle property with one image name

Requires

Any view controller presenting this view must conform to CSBUnwindableViewController
